# Annotated dictionary of Kouyoumdjian 1970 - "A Comprehensive Dictionary: Armenian-English"

Years ago, I started annotating entries from Kouyoumdjian 1970 - "A Comprehensive Dictionary: Armenian-English". Over the years, the contents were used to design the [Apertium Western Armenian transducer](https://github.com/apertium/apertium-hyw), the ongoing DALiH project for designing a Western Armenian corpus, and my own personal project on writing a [Western Armenian grammar](https://github.com/jhdeov/WesternArmenianGrammar). 

This public-facing copy provides around 63k lemmas. As of now, I have provided the following corrected or partially-corrected annotations:

* Phonological transcription: One using Armenian dialectological symbols, and one in IPA
* Underlying Representation in Armeinan orthography, with undoing of phonological processes and the addition of morphological boundaries
* Information of grammatical class information, including regularity, conjugation classes, and a note on the class number of verbs from the [Hay-Pay book](https://github.com/jhdeov/ArmenianVerbs)

It's still a work in progress because I have to double-check all the class information as I start writing the morphology chapters of my grammar. But I figured I might as well keep a current copy here in case anyone could benefit from it while I slowly work.
